# Hangman-game
A classic Hangman word-guessing game built in Python with ASCII art, color output, and multiple word categories.

## Feautures
- 📍 ASCII art hangman (6 stages)
- 🎨 Colorful output with colorama
- 📖 Category hints with emojis
- 👤 Name confirmation before playing
- 📝 Tracks guessed letters
- 🔄 Play again option
- ✅ Input validation
